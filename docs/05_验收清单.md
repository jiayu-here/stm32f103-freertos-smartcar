# 完整验收清单

## A. 软件构建

- [x] C8T6 使用 63 KB 程序区成功编译、链接
- [x] RCT6 使用 254 KB 程序区成功编译、链接
- [x] 最后一页 Flash 未被程序段占用
- [x] ELF、HEX、BIN、MAP 均生成
- [x] PID 和全部文本命令主机测试通过
- [x] SVC、PendSV、SysTick、USART1、EXTI4 中断符号存在

## B. 烧录和启动

- [ ] ST-Link 能识别准确芯片型号
- [ ] 写入、verify、reset 全部成功
- [ ] 串口出现 `SMARTCAR READY`
- [ ] VERSION、STATUS、CONFIG、DIAG 都有合理回复
- [ ] OLED 两个页面正常切换

## C. 电机和编码器

- [ ] DRIVE 正值时两轮均前进
- [ ] 正向 RPM 均为正
- [ ] 左右轮 CPR 已按四倍频机械轮一圈标定
- [ ] 30/60/100 RPM 稳态误差可接受
- [ ] 正转、反转和差速无持续振荡
- [ ] RAMP 能降低启动冲击

## D. 里程计和定量运动

- [ ] MOVE 100 的实测距离满足要求
- [ ] TURN 90、-90、360 的方向和误差满足要求
- [ ] ODOM X/Y/heading 变化方向正确
- [ ] RESETODOM 只在停车时成功
- [ ] 8 段路线可编辑、查看、运行和取消
- [ ] 急停会取消正在运行的路线

## E. 自动模式

- [ ] 五路红外位序和有效电平正确
- [ ] TRACK 能通过直线和弯道
- [ ] 丢线后朝最后偏差方向搜索
- [ ] HC-SR04 10/30/100 cm 数据合理
- [ ] 舵机左右扫描方向正确且不会造成复位
- [ ] AVOID 能选择空间较大的方向
- [ ] FUSION 遇障后能继续寻找线路
- [ ] CRUISE 前方过近会停车

## F. 参数 Flash

- [ ] 修改 PID/WHEEL/RAMP 后 CONFIG 显示正确
- [ ] SAVE 返回 OK SAVED，sequence 增加
- [ ] 重新上电后参数保持
- [ ] DEFAULTS 只改变 RAM，未 SAVE 前复位仍回到旧参数
- [ ] 反复保存超过一页槽数后仍能加载最新记录
- [ ] 写入过程中断电不会加载半写记录

## G. 安全保护

- [ ] 手动指令停止约 1 秒后停车
- [ ] PA15 按键立即停车并锁存
- [ ] STOP 软件急停锁存，松键/CLEAR 后才恢复
- [ ] 低于低电阈值停车，高于恢复阈值才解除
- [ ] 堵转条件持续达到时间后出现 fault 0x20
- [ ] 故障发生时旧目标和路线被清除，不会在 CLEAR 后突然恢复
- [ ] 蜂鸣器和 LED 状态正确
- [ ] 人为阻塞任务后 IWDG 能复位

## H. 资源和老化

- [ ] DIAG 的 heap_min 保留足够余量
- [ ] 七个 stack_words 均不接近 0
- [ ] 连续运行 30 分钟无异常复位
- [ ] 电机驱动、DC-DC、导线和电池不过热
- [ ] 电机干扰不会造成编码器跳变、ADC异常或串口断连
