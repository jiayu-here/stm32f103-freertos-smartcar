# v2.0.0 — C8T6 增强版

## 新增能力

- C8T6 安全范围内的 Flash 参数日志、CRC32 和磨损控制。
- 在线 PID、速度、斜坡、循迹、里程计、电池和堵转参数配置。
- 双编码器里程计、定距 MOVE、定角 TURN、八段路线队列。
- 堵转保护、急停、低电、命令超时、任务健康和 IWDG。
- FreeRTOS 堆/栈诊断、复位原因和 RAM 故障历史。
- 全部硬件接线、组件、软件架构、协议、标定、操作和验收说明。

## 构建产物

Release 附件包含：

- `smartcar_C8.hex` / `smartcar_C8.bin`
- `smartcar_RCT6.hex` / `smartcar_RCT6.bin`
- `SHA256SUMS.txt`

选择与实际 MCU 一致的镜像。C8T6 固件使用 63 KB 程序区，最后 1 KB 预留给参数记录。

## 验证

- Arm GNU Toolchain 14.2.1：C8T6 和 RCT6 完整构建通过。
- PID/协议主机测试、引脚/文档/Flash 分区一致性检查通过。
- 硬件实测仍需按 `docs/05_验收清单.md` 完成。
