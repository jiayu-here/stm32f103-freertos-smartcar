# 硬件接线、供电与引脚说明

## 1. 适用主控

相同 GPIO 名称适用于 STM32F103C8T6 和 STM32F103RCT6。不同开发板的排针位置可能完全
不同，必须按板上丝印或原理图寻找 PA0、PB12 等引脚，不要照抄物理排针序号。

固件保留 PA13/PA14 作为 SWD，并关闭 JTAG 释放 PB3/PB4。PA15 同样在关闭 JTAG 后作为
急停按键输入。

## 3. TB6612 和电机

```text
STM32 3.3V -> TB6612 VCC
电机电源   -> TB6612 VM
公共地     -> TB6612 GND、STM32 GND、电池负极
AO1/AO2    -> 左电机
BO1/BO2    -> 右电机
```

VM 附近并联至少 470 uF 电解电容和 100 nF 陶瓷电容。电机堵转电流必须小于驱动器和电源
能安全提供的电流。软件堵转检测不能保护错误接线或电源短路。

## 4. 编码器

编码器 A/B 相分别连接定时器两通道。若输出是开漏，使用 3.3 V 上拉。正向运行时串口中的
RPM 必须为正；符号错误时修改 `BSP/Inc/board.h` 中：

```c
#define LEFT_ENCODER_SIGN  (1.0f)
#define RIGHT_ENCODER_SIGN (1.0f)
```

只把需要反向的一侧改为 `-1.0f`。编码器每圈计数通过 `WHEEL` 命令设置。

## 5. HC-SR04 电平保护

HC-SR04 使用 5 V，Echo 也是约 5 V，不能直接进入 STM32：

```text
HC-SR04 Echo ---- 10k ----+---- PA4
                          |
                         20k
                          |
                         GND
```

PA4 电压约为 5 × 20/(10+20) = 3.33 V。Trig 的 3.3 V 高电平通常可被模块识别。

## 6. SG90 舵机

SG90 红线接独立 5 V，棕/黑线接公共地，黄/橙线接 PA8。建议 5 V 降压模块持续电流至少
1 A。禁止从 STM32 3.3 V 或小电流板载稳压器给舵机供电。

## 7. OLED 和 HC-05

- OLED：VCC 接 3.3 V，GND 共地，SCL 接 PB8，SDA 接 PB9。
- HC-05：VCC 按模块底板要求连接，GND 共地，TXD 接 PA10，RXD 接 PA9。
- 正常通信波特率必须配置为 115200；AT 模式波特率可能不同。

## 8. 电池检测

```text
电池正极 ---- 100k ----+---- PA2
                       |
                      33k
                       |
                      GND

PA2 ---- 100nF ---- GND
```

默认倍率 4.0303、低电 6.60 V、恢复 6.90 V，适用于常见 2S 电池。修改命令：

```text
BATTERY 6.60 6.90 4.0303
SAVE
```

若使用其他串数，必须同时确认 ADC 最大输入电压不超过 3.3 V。

## 9. 电源分配建议

```text
2S 电池
  +--> TB6612 VM --> 两个电机
  +--> 5V DC-DC --> SG90、HC-SR04、需要 5V 的模块
  +--> 3.3V 稳压 --> STM32、OLED、TB6612 VCC、3.3V 传感器
所有 GND 相连
```

电机线和 PWM 大电流回路应远离 PA2 电池采样线、编码器线和 SWD 线。

## 10. 首次通电顺序

1. 只接主控和 ST-Link，验证烧录。
2. 接 OLED、HC-05、红外和超声波，检查 STATUS。
3. 架空车轮，接 TB6612 和编码器，小速度检查方向。
4. 使用独立 5 V 接 SG90，检查扫描方向和供电稳定性。
5. 完成参数标定后再落地运行。

详细操作见 `docs/09_操作步骤_从零到运行.md`。
