# 蓝牙与串口命令完整说明

## 1. 接口格式

- USART1：PA9 TX、PA10 RX
- 115200 baud、8 数据位、无校验、1 停止位
- ASCII 文本，不区分大小写
- 每条命令以 CR、LF 或 CRLF 结束
- 成功通常返回 `OK`，失败返回 `ERR`，自动事件以 `EVENT` 开头

## 2. 运行控制

| 命令 | 参数范围 | 说明 |
|---|---|---|
| `MODE MANUAL` | 固定名称 | 手动模式，等待 DRIVE |
| `MODE TRACK` | 固定名称 | 自动循迹 |
| `MODE AVOID` | 固定名称 | 自动扫描避障 |
| `MODE FUSION` | 固定名称 | 循迹与避障融合 |
| `MODE CRUISE` | 固定名称 | 前方安全时直行 |
| `DRIVE L R` | 各 -180..180 RPM | 进入手动模式并设置左右速度 |
| `SPEED rpm` | 20..150 | 自动模式和定距运动基础速度 |
| `STOP` | 无 | 软件急停并锁存 |
| `CLEAR` | 无 | 松开物理按键后清除急停和堵转锁存 |
| `CANCEL` | 无 | 正常取消定距/定角/路线并回到 MANUAL |

手动运动必须周期性发送 DRIVE，超过约 1 秒没有新 DRIVE 会触发超时停车。

## 3. 定距、定角和里程计

| 命令 | 示例 | 说明 |
|---|---|---|
| `MOVE cm` | `MOVE 100` | 前进 100 cm；负数后退；绝对值不超过 1000 |
| `TURN deg` | `TURN 90` | 左转 90°；负数右转；绝对值不超过 1000 |
| `ODOM` | `ODOM` | 输出 X、Y、航向、左右累计距离 |
| `RESETODOM` | `RESETODOM` | 完全停止时把里程计清零 |

完成时会返回：

```text
EVENT MOVE DONE
EVENT MOTION IDLE
```

## 4. 路线队列

| 命令 | 说明 |
|---|---|
| `QMOVE cm` | 向 RAM 路线加入定距段 |
| `QTURN deg` | 向 RAM 路线加入转角段 |
| `ROUTE` | 列出路线；当前段标记 ACTIVE |
| `RUN` | 完全停止时从第 1 段运行路线 |
| `CLEARROUTE` | 未运行时清空路线 |

最多 8 段。例如执行矩形路线：

```text
CLEARROUTE
QMOVE 100
QTURN 90
QMOVE 60
QTURN 90
QMOVE 100
QTURN 90
QMOVE 60
RUN
```

路线存放在 RAM，复位后丢失；参数 Flash 不保存路线。

## 5. PID 和标定参数

| 命令 | 范围 | 说明 |
|---|---|---|
| `PID kp ki kd` | Kp/Ki 0..20，Kd 0..5 | 同时更新左右 PID |
| `RAMP rpm_s` | 20..1000 | 目标速度每秒最大变化 |
| `LINEGAIN gain` | 1..40 | 红外偏差到左右差速的增益 |
| `WHEEL diameter track cpr` | 2..20 cm，5..50 cm，20..20000 | 轮径、有效轮距、四倍频 CPR |
| `BATTERY low recover ratio` | 合理电压/倍率范围 | 低电、恢复、分压倍率 |
| `STALL pwm rpm ms` | 20..100%，0.5..30 RPM，200..5000 ms | 软件堵转条件 |

参数修改仅在 RAM 中生效，返回会提示 `USE SAVE`。

## 6. 参数存储

| 命令 | 说明 |
|---|---|
| `CONFIG` | 显示当前全部参数和 Flash 序号 |
| `SAVE` | 完全停止时追加保存一条参数记录 |
| `LOAD` | 完全停止时重新加载 Flash；无记录则使用默认值 |
| `DEFAULTS` | 完全停止时恢复 RAM 默认值，不自动写 Flash |

推荐流程：调整参数 -> 测试 -> `CONFIG` 核对 -> `MODE MANUAL` 并停车 -> `SAVE` ->
复位 -> `CONFIG` 验证。本文中“完全停止”均表示 MANUAL 且目标/电机输出已经为零。

## 7. 状态和诊断

| 命令 | 说明 |
|---|---|
| `STATUS` | 模式、目标/实际 RPM、距离、电池、姿态、红外和 fault |
| `DIAG` | 当前/最小堆、七任务栈高水位、复位原因 |
| `LOG` | 最近 8 次 RAM 故障变化 |
| `VERSION` | 固件版本 |
| `HELP` | 命令摘要 |

状态示例：

```text
STAT mode=5 target=42.0,42.0 rpm=40.8,41.3 dist=85.2 bat=7.61 pose=63.4,1.2,2.1 ir=04 fault=0
```

## 8. fault 位图

| 位值 | 含义 |
|---:|---|
| 0x01 | 低电 |
| 0x02 | 急停锁存 |
| 0x04 | 手动命令超时 |
| 0x08 | 任务健康异常 |
| 0x10 | 预留传感器故障 |
| 0x20 | 电机堵转锁存 |

## 9. reset 位图

| 位值 | 含义 |
|---:|---|
| 0x01 | NRST/引脚复位 |
| 0x02 | 上电或掉电复位 |
| 0x04 | 软件复位 |
| 0x08 | IWDG 复位 |
| 0x10 | WWDG 复位 |
| 0x20 | 低功耗复位 |

多个位可能同时存在，应结合电源和运行日志判断。
